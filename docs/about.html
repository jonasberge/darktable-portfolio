About this website

This is a preview of what my photography portfolio could look like in the future, the source code of this prototype can be found on GitHub here. The main aspect of this is that I wanted to generate my portfolio with one click from the photos I am managing in the open-source photo editing software Darktable. The code behind this website accesses my Darktable library, detects which images I tagged for use in my portfolio and generates static HTML output, which enables me to use Darktable like a Content Management System with a very capable user interface. For managing my portfolio I do not have to leave my editing software.

All images are my own. Copying and/or redistributing is not allowed.
